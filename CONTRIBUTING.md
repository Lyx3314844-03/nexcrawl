# Contributing to NexCrawl

We love your input! We want to make NexCrawl the best web intelligence framework in the world.

## How to Contribute

### 1. Reporting Bugs
- Use the GitHub Issue Tracker.
- Provide a minimal reproduction script.
- Include logs (with sensitive tokens redacted).

### 2. Suggesting Features
- Open an issue labeled `enhancement`.
- Describe the business scenario and technical feasibility.

### 3. Code Submissions
- Fork the repository.
- Create a feature branch (`git checkout -b feat/my-new-feature`).
- Ensure all tests pass (`npm test`).
- Commit your changes with clear messages.
- Submit a Pull Request.

## Development Setup

```bash
# Clone the repo
git clone https://github.com/Lyx3314844-03/nexcrawl.git

# Install dependencies
npm install

# Run tests
npm run test:reverse
```

## Security Policy
Please do not report security vulnerabilities through public issues. Email them to the maintainers or use GitHub's private reporting feature.
